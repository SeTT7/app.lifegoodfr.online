After Module I, move into The Sacred Chants for daily practice. Then read The Dead Sea Revelation, and finally integrate everything through the Divine Body Ritual.

There is no fixed timeline — this is a devotional kit, not a course with a finish line. Most readers take 1 to 2 weeks to read Module I, then return to it many times. Treat the kit as a library you live with, not a list to finish.

The recommended order is the one shown in the home menu, and it produces the deepest results. Module I is the only one we strongly suggest you start with — it prepares you to receive everything else.

After that, you are free to revisit any module in any order, as your soul calls for it.
